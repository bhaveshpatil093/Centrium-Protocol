## 🚩 Problem Statement
> 💥 Web3 information is currently scattered across hundreds of platforms, making it difficult for users to find reliable, up-to-date resources. Newcomers face steep learning curves while experts struggle to share knowledge effectively.

## 💡 Solution
Centrium Protocol provides:
- 🏗️ Unified platform for structured Web3 knowledge
- 🧩 Modular content organization system
- 🏆 Incentive mechanisms for quality contributions
- 🔗 Cross-platform resource aggregation
- 📊 Reputation system for content validation

## 🛠️ Installation

### Prerequisites
- Node.js v16+
- npm v8+
- Hardhat
- MetaMask (Browser Extension)

### Setup Instructions
  1. **Clone repository**:
     
     ```bash
     git clone https://github.com/bhaveshpatil093/Centrium-Protocol.git

  2. **Install dependencies**:
     
     ```bash
     cd Centrium-Protocol
     npm install

  3. **Set up environment variables**:
     
     ```bash
     cp .env.example .env

  4. **Start the development server**:

     ```bash
     npm run dev

## 🏁 Getting Started

### 📋 Prerequisites
- Node.js v16+
- npm v8+
- MetaMask extension installed
- Hardhat (for smart contract interactions)

### 🖥️ Local Development Setup

  1. **Clone the repository**:

     ```bash
     git clone https://github.com/bhaveshpatil093/Centrium-Protocol.git

  2. **Navigate to project directory**:

     ```bash
     cd Centrium-Protocol

  3. **Install dependencies**:

     ```bash
     npm install

  4. **Set up environment variables**:

     ```bash
     cp .env.example .env
     # Add your keys to the .env file

  5. **Start local blockchain**:

     ```bash
     npx hardhat node

  6. **Deploy contracts (in a separate terminal)**:

     ```bash
     npx hardhat run scripts/deploy.js --network localhost

  7. **Start frontend**:

     ```bash
     npm run dev

## 🏁 Getting Started

1. **Access the Platform**: Visit `http://localhost:3000` in your browser
